# **CREATE GIF WITH PYTHON**

## **📌 Description**

This project focuses on creating GIF with multiple images using python and imageio library

## **🚀 Features**

- Creates GIF from multiple images
- simple to use 
- beginner-friendly project

## **🛠️ Requirements**

- Python 3
- imageio - pip install imageio

## **📂 File structure**

- Main script - create_gif.py
- images - Add your images 
- Example out put - Generated.gif

## **▶️ How to Run**

1. Add your images to the same folder as the main script
2. Run the script - python3 create_gif.py
